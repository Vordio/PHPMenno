Inbouwen

Het inbouwen van dit systeem is eigenlijk heel makkelijk.
Je hebt uiteraard al een eigen index.html of index.php.
Anders kan je gewoon kladblok openen en precies doen wat ik nu zeg :

-Plak deze code helemaal bovenaan je index.php
<?php
/// Start van installatie check ///
if(!isset($var) && file_get_contents("config.php") == "") {
include('pagina/maakconfig.php');
die();
}else{
require_once('class/head.class.php');
}
/// Einde van installatie check ///
?>

-Op de plek waar je je menu wilt hebben.
Plak dit script:
<?php
include('menu.php');
?>


-Op de plek waar je je algemene tekst wilt hebben. Dus alle tekst van elke pagina.
Plak dit script:
<?php
include('pagina.php');
?>

Plak dit script:
<?php
include('pagina/balk.php');
?>

Systeem Installeren

Het installeren is heel simpel,
Zodra je alle bestanden hebt geupload kan je gewoon index.php laden en de installatie komt vanzelf te voorschijn.
Vul hier dus je MySQL gegevens in die je van je host heb gekregen:
-Gebruikersnaam
-Wachtwoord
-Databasenaam
-Host (meestal localhost)

Zodra je de hele installatie hebt uitgevoerd zou je systeem perfect moeten werken.
Belangrijk!!! je moet de bestanden "config.php", "pagina/maakconfig.php" en "install.php" CHMODDEN naar 777 voor een optimale werking!!!

Als je systeem nu nog steeds niet werkt.
Neem contact op met de developers van dit systeem.
of ga naar habbers menno ledensysteem topic!
