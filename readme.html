Problemen

Heb je een probleem met grote lettertype's?
Dan komt het door CSS
Open je index.php en plak nèt boven </head> deze code:

<style type="text/css">
table,tr,td,th {
   font-size: 11px;
}
</style>


Inbouwen

Het inbouwen van dit systeem is eigenlijk heel makkelijk.
Je hebt uiteraard al een eigen index.html of index.php.
Anders kan je gewoon kladblok openen en precies doen wat ik nu zeg :

-Plak deze code helemaal bovenaan je index.php
<?php
/// Start van installatie check ///
if(!isset($var) && file_get_contents("config.php") == "") {
include('pagina/maakconfig.php');
die();
}else{
require_once('class/head.class.php');
}
/// Einde van installatie check ///
?>

-Op de plek waar je je menu wilt hebben.
Plak dit script:
<?php
include('menu.php');
?>


-Op de plek waar je je algemene tekst wilt hebben. Dus alle tekst van elke pagina.
Plak dit script:
<?php
include('pagina.php');
?>



-Op de plek waar je je berichten balk wilt hebben.
Plak dit script:
<?php
include('pagina/balk.php');
?>



-Op de plek waar je je Laatste 5 Geregistreerde gebruikers wilt hebben
Plak dit script:
<?php
include('statistieken/laatste_5.php');
?>



-Op de plek waar je je Top 5 rijkste leden wilt hebben
Plak dit script:
<?php
include('statistieken/top_10_muntjes.php');
?>



-Op de plek waar je je nieuws headlines wilt hebben
Plak dit script:
<?php
include('statistieken/nieuwsheadlines.php');
?>



-Op de plek waar je je Top 5 gebruikers wilt hebben.
Plak dit script:
<?php
include('statistieken/top_5.php');
?>



-Op de plek waar je je aantal geregistreerde gebruikers wilt hebben.
Plak dit script:
<?php
include('statistieken/aantal_leden.php');
?>



-Op de plek waar je 5 willekeurig gekozen gebruikers wilt hebben.
Plak dit script:
<?php
include('statistieken/random_5.php');
?>



-Op de plek waar je de laatste nieuwsberichten wilt hebben.
Plak dit script:
<?php
include('statistieken/laatste_nieuws.php');
?>



-Op de plek waar je het aantal online gasten en leden wilt hebben.
Plak dit script:
<?php
include('statistieken/online.php');
?>

<?php if(!isset($_SESSION['id'])) { ?>

<?php }else{ ?>

<?php } ?>

Systeem Installeren

Het installeren is heel simpel,
Zodra je alle bestanden hebt geupload kan je gewoon index.php laden en de installatie komt vanzelf te voorschijn.
Vul hier dus je MySQL gegevens in die je van je host heb gekregen:
-Gebruikersnaam
-Wachtwoord
-Databasenaam
-Host (meestal localhost)

Zodra je de hele installatie hebt uitgevoerd zou je systeem perfect moeten werken.

Als je systeem nu nog steeds niet werkt.
Neem contact op met de developer van dit systeem.
